# 活力二人组

熊猫和 SQLite3 的另一个快速定义。

Pandas 是一个非常受欢迎的 Python 库，它允许您将文件永久存储到数据帧中，然后可以很容易地对其进行操作。它还与多种可视化兼容，允许对大数据进行交互式探索。

SQLite3 是从 *Python 2.5* 开始就包含在 Python 库中的一个模块。它添加了一个温和的关系数据库元素，使您能够根据传统的 SQL 语法编写查询，并相对容易地获取数据。
