# **基本思路**

基本级别的区块链是考虑数据库的一种新方式。使区块链不同于传统数据库的是完成区块链资产记录的安全包装，使该记录不可更改。此外，区块链技术通常是分散的；这意味着资产记录分散在许多计算节点上，而不是集中在一个公司。

这些独特的原则意味着区块链可以提供一种前所未见的新的数字记录保存方法。区块链正在提供增强的安全性，并且没有潜在对手利用和攻击的单点故障。

我认为区块链的公共性质导致许多公司担心使用它作为解决方案。这种透明度是经营业务的一种新思路。但是你可以有一个公共的店面，可以这么说，并对个人记录保密。不要因为这些独特的特征就不把区块链作为一个解决方案。事实上，正是这些独特的特点使区块链在为商业的未来发展新的商业模式方面具有创新性和吸引力。

# **区块链 vs 分布式账本**

在区块链的对话中，你可能也听说过分布式账本这个术语。不过，值得注意的是，术语“分布式分类帐”和“区块链”不应互换使用，因为并非所有分布式分类帐都是区块链。有些人交替使用这两个术语，这似乎有几个原因；

*   区块链是一项新兴技术，这使得它有不充分的定义
*   与区块链和加密货币骗局相关的负面含义
*   区块链可以实现去中心化的经济未来
*   分布式分类帐不是区块链，但是区块链可以形成分布式分类帐

# 区块链试图解决什么问题？

区块链并不是为了解决传统的数据库问题。有许多合理的数据库解决方案可以满足公司对集中式数据库的需求。区块链试图解决的是一种新的解决方案，可以有效地管理组织或个人之间的信任，而不需要传统上有保证信任的中间公司。

中央公司的一个例子是结算金融交易的票据交换所。清算所提供两个未知方之间的信任。或者，如果你喜欢读书，另一个日常的中心机构是图书馆。

区块链让人们和他们的日常交易相互进行，并通过在无许可的数字状态下通过规则保护交易，在数字自治系统中建立信任。

# 关键要点

一个可信的区块链应该以这三个重要的基本原则为核心；

1.  **防篡改**——存储在区块链上的每一笔交易都使用独特的数学代码(密码术)进行保护
2.  透明性——一个单一的公共分类账，每个人都可以阅读，每个人都可以写，但没有人可以改变。
3.  **信任** —数字资产不存储在中央数据库中，而是分布在使用最高安全性的全球分类账中。没有正确的协议就不能改变分类帐条目，从而在没有集中平台的情况下创建虚拟信任。

总的来说，区块链的创造和创新及其思想试图在一个不信任的社会中优雅地建立信任。为下一代升级我们社会运行的系统。
