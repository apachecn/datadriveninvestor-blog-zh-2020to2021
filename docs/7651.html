你应该加入观察名单的ETF

能源行业是一个目前正在我们面前发展的行业。在很大程度上，该领域最大的公司仍然是历史悠久的石油公司，如雪佛龙、埃克森美孚和康菲石油公司。 XLE ，能源精选SPDR ETF，截至2020年12月16日，2020年下跌约-29%。

幸运的是，在能源领域有一个快速增长的部分。这部分是清洁和可再生能源。随着全球清洁和可再生能源计划的增多，以下3只ETF是你在2021年应该关注的:

1.第一信托纳斯达克清洁优势绿色能源指数基金( QCLN

2.景顺太阳能ETF ( 谭)

3.景顺威尔清洁能源ETF ( PBW )

1。第一信托纳斯达克清洁边缘绿色能源指数基金(【QCLN】)

QCLN是一个ETF，旨在跟踪纳斯达克清洁能源指数的投资结果，该指数旨在跟踪在美国上市的小型、中型和大型清洁能源公司的表现。QCLN中的公司有 NIO 、 SolarEdge Technologies 、 Enphase Energy 和 Tesla 。

截至2020年12月16日，QCLN的价格为64.80美元，费用率为0.60%。QCLN今年迄今的回报率约为163%。

2。景顺太阳能ETF ( 【谭】 )

TAN是一只ETF，旨在跟踪 MAC全球能源指数的投资结果，该指数旨在为在发达市场交易所上市的公司提供投资机会，这些公司的很大一部分收入来自太阳能行业。在谭的公司有 SolarEdge Technologies 、 Enphase Energy 、信义太阳能控股、 First Solar 。

截至2020年12月16日，TAN的价格为92.12美元，费用率为0.71%。今年以来，谭的回报率约为202%。

3。景顺威尔清洁能源ETF()

PBW是一只ETF，旨在跟踪 WilderHill清洁能源指数的投资结果，该指数由美国从事清洁能源发展和保护业务的上市公司股票组成。PBW的公司有燃料电池能源、 NIO 、闪充、插电。

截至2020年12月16日，PBW的价格为94.48美元，费用率为0.70%。PBW今年迄今的回报率约为177%。

如你所见，QCLN、谭和在2020年远远超过了。在未来十年，看到更多清洁和可再生能源公司在能源领域占据更大的市场份额并不奇怪。
