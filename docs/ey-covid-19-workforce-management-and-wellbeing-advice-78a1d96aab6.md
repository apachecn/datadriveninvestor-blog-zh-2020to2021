# 格雷厄姆·里德

最近发生了很多变化。对于许多人来说，这是非常令人不安的一周，无论是在个人生活还是商业生活中，许多公司执行在家工作的规定，酒吧和餐馆关门，失业人数急剧增加。一些国家实行了广泛的旅行禁令和封锁，而其他国家则大幅增加了应对措施。

总的来说，在任何层面上，这都是一个很大的问题，我们应该花一点时间来认识到随着情况的变化，全球各国政府和卫生工作者所做的令人难以置信的工作。

EY 最近几天一直在与西班牙和意大利的同事密切合作，他们将这场危机的阶段描述为:

1.  无知。
2.  恐慌。
3.  接受。
4.  分辨率。

在我们新现实的第一周，我们似乎正从恐慌走向接受，这个现实看起来将在可预见的未来持续下去。但是，尽管情况如此，生活和业务必须继续，有一件事应该鼓励我们前进，那就是，对大多数人来说，适应新的工作实践远没有最初预期的那么具有破坏性。

# 尼亚姆·奥贝恩

我们都在工作和生活中经历着这样一种创伤性的变化，习惯于充满电话的房子，学习如何有效地使用我们的静音按钮，确保我们在语音而不是视频上——而所有这一切都是在我们的孩子接受家庭教育或至少在家的时候。我们都对学校和老师产生了新的欣赏。

但是，新冠肺炎疫情带来的快速升级的挑战确实为组织提出了一些需要解决和评估的关键业务和人员问题。我们所有的劳动力都受到了危机的影响，包括行动受限、焦虑加剧、隔离和学校关闭。

**现在领导者面临的关键问题是:**

1.  你的员工安全吗？如果是，你是如何维护的？
2.  他们是否可以执行关键的业务职能？
3.  你是否有足够的弹性和能力在短期和中期内维持你的组织，优先部署你的劳动力，以确保业务连续性和最大限度地提高效率？
4.  你还会继续这种交易心态吗？

疫情与经济衰退截然不同——变化的速度非常快。随着经济衰退，你会看到拐角处，但远不如疫情。上周，[世卫组织的马丁·瑞恩博士说:](https://www.who.int/docs/default-source/coronaviruse/transcripts/who-transcript-emergencies-coronavirus-press-conference-full-13mar2020848c48d2065143bd8d07a1647c863d6b.pdf?sfvrsn=23dd0b04_2)

> “速度胜过完美，我们目前面临的社会问题是，每个人都害怕犯错误，每个人都害怕错误的后果。但最大的错误是不行动，最大的错误是被失败的恐惧所麻痹。”

组织需要行动起来保护他们的员工，并且需要迅速行动起来。那么，我们需要做什么？

**度过危机:如何在中期内应对**

危机的短期影响将持续 2-3 个月，随着组织开始调整，将更加关注业务连续性。运用以人为本的思维方式，领导者可以建立弹性。我们需要考虑 4 件事:

1.  识别关键的业务流程，更重要的是，识别执行这些流程的人员。
2.  创建安全的工作环境和有效的远程工作实践，实现业务连续性。
3.  进行继任规划和短期回填。
4.  制定符合组织价值观和文化的清晰的沟通策略。

**关键任务:业务连续性**

1.  *识别关键流程和人员* —进行快速诊断以识别关键流程和关键人员及资源；确定每个人需要付出的努力程度；如果可能的话，考虑工作转移和迁移某些关键流程。
2.  *确保任务的有效委派*——跨资源的工作量分担，以及员工或供应商的交叉培训；并审查和更新操作程序，以确保持续有效。
3.  准备关闭办公室 —确定哪些服务和流程可以暂停；并寻求法律支持以确保法规遵从性。

**好的领导**

此时，良好的领导力和榜样作用比以往任何时候都更加重要。在沟通、政策和行动中展现灵活性和同理心将:

*   释放团队合作、创新和协作的潜力。
*   为文化创造一个跳板，使一个组织与众不同。
*   优先考虑员工福利。
*   避免无意识的偏见。
*   防止工作场所的歧视。

现在是将这些价值观植入组织的最佳时机。

**创造安全的工作环境，确保连续性**

1.  *创建* —确保现场遵守卫生部和健康安全环保部的建议；规划您的技术基础设施如何支持业务；和管理包容性。
2.  *准备* —在家工作已经成为新的规范，所以考虑:额外的 IT 设备和支持；远程工作工具和能力；远程工作和福利指南和提示；和弹性工作制，以方便照顾孩子、做茧自缚等。
3.  *适应* —重要的是:监控因需求增加而产生的技术影响；把时间用于自我发展和幸福；并确保反馈渠道畅通，以便我们都能学习并继续发展。

**中国的经验教训**

中国对危机反应的有效性部分归功于他们对数字工具的使用。该国政府要求其最大的科技公司支持某种“战斗号令”，阿里巴巴、百度和腾讯等公司的回应是免费将他们最好的创新放在桌面上。我们需要在其他地方看到更多这种利他主义。

**清晰的沟通策略**

在任何业务中断期间，内部和外部沟通都至关重要，尤其是疫情。没有剧本，这意味着反应是即兴和被动的。危机管理团队负责监控事件并促进持续沟通。组织需要确保他们有更新的联系信息，以便他们可以快速联系到员工。

1.  *实用性* —在遵循相关指导的同时，将通信置于危机的中心；围绕业务连续性和工作人员安全和安保开展宣传活动，确保所有利益攸关方都参与到宣传计划中；对员工保持坦诚，定期提供反馈。
2.  *沟通渠道* —利用电子邮件和公告等传统渠道；使用协作工具和社交媒体来支持新的工作规范；把领导者放在沟通的最前面；和测试紧急通知系统。
3.  *活出你的文化*——你的文化需要比以往更响亮、更清晰；将文化和价值观置于所有信息和行动的核心；旨在传达自信、同情、人性和现实主义。

**员工福利和心理健康**

领导者应该:

*   确保员工知道在哪里可以找到信息。
*   为对精神和身体健康的影响规划资源。
*   鼓励关注社区、团队和自我发展。
*   让员工能够表达他们的担忧，以及他们是如何被倾听的。
*   从最高层的正确语气开始，确保没有偏见和无意识的偏见。

# 蒂姆·伯金

在这个不确定的时期，我们试图优化劳动力部署，因此面临着许多明显的挑战。然而，也有一些机会是由以更客观和诚实的方式看待我们的组织的需要所驱动的。

人们一致认为，我们正在应对的可能是一个长期的中断，这种中断会不断演变和变化。为了有效应对，尽早建立正确的基础至关重要。这样做有四个步骤:

1.  *部署一致的框架* —实施集成一致的框架为有效的劳动力连续性规划奠定了基础。
2.  *让业务领导和经理参与* —除了让他们参与劳动力连续性规划之外，支持业务领导和经理积极主动地进行批判性思考，对于激励行动至关重要。
3.  *收集洞察力，为短期、中期和长期规划提供信息* —在线调查可用于清楚了解整个企业的劳动力挑战；使用定制的仪表盘进行分析和可视化也将有助于确保快速分析。
4.  *定期重复，以实现敏捷性和面向未来的分析* —定期重复数据分析有助于确保业务做好准备，以应对不断变化的环境；并且可以进行情景模拟以评估可能的新冠肺炎结果的影响。

**EY 劳动力连续性框架**

要使用这一框架并使其在业务中发挥作用，必须采取 3 个关键步骤:

1.  评估对业务、客户和员工的影响。
2.  优先考虑劳动力部署，以最大限度地提高效率和效力，重点关注以下任务:*关键* *任务* —这些任务必须承担并完成，以确保业务连续性；*必需的* *任务* —这些对客户体验和业务表现有重大影响；或者*可选任务* —这些任务对业务改进很重要，但对运营绩效并不关键。
3.  最小化干扰的行动计划:*响应* —需要采取什么行动来完成关键的、必需的或可选的任务？；*维持* —在没有未知时间表的情况下，需要采取什么措施来维持响应？；以及*复位*——需要采取什么行动来准备最终复位？明天需要做哪些铺垫？
