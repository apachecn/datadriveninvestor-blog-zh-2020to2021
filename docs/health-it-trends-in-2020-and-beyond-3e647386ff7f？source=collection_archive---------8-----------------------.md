# 2020 年及以后的医疗 IT 趋势

> 原文：<https://medium.datadriveninvestor.com/health-it-trends-in-2020-and-beyond-3e647386ff7f?source=collection_archive---------8----------------------->

![](img/6568f7c59308cc3aea720699d0016678.png)

世界上没有一个部门不受技术及其不断发展的影响。为什么医疗保健应该例外？毕竟，人类几千年来一直在利用技术改善健康。这肯定不会很快改变。

当谈到新兴技术趋势时，经常会用到很多流行词汇。许多人抛出“区块链”或“物联网”这样的短语谈到医疗保健 it，在可预见的未来，真正定义未来的趋势是什么？以下是 2019 年医疗 IT 的五大趋势。

# **虚拟保健**

我们都习惯于去看医生，我们通常会跳上一辆车，开到我们医生的办公室，无论是标准检查还是更严重的事情。然而，虚拟护理的概念将在未来几年内变得更为人们所接受。技术现在允许[远程病人监控](https://healthtechmagazine.net/article/2018/11/healthcare-organizations-tap-remote-patient-monitoring-manage-conditions-real-time-perfcon)，这可以帮助医生在办公室之外跟踪他们的病人。

从任何意义上来说，这可能都不是“规范”，但我们可以预计将有更多的虚拟护理应用程序供患者使用，这可能意味着与医生打交道比以往任何时候都更便宜、更方便。

患者希望获得更多的医疗保健，他们希望获得临床环境之外的服务。你可以期待虚拟护理的概念在 2019 年及以后变得更加流行。电子病历(EMR)公司也将在管理患者数据方面发挥重要作用。

# **关注网络安全**

网络安全仍将是医院和医疗保健提供商的重中之重，这部分是因为网络安全在一般意义上比以往任何时候都更加重要。具体来说，到 2021 年，网络犯罪预计会让全球的公司损失惊人的 6 万亿美元。随着威胁的增加，医疗保健行业的组织将有望投资于网络安全。

没有多少记录比医疗数据更加隐私和敏感，因此这是一个非常现实的问题。还应该注意的是，医疗保健黑客[正在上升](https://www.dotmed.com/news/story/49380)，这表明网络犯罪分子可能会在未来更直接地瞄准医疗保健部门。

# **人工智能**

有许多人认为人工智能是一种会夺走人类工作的力量。然而，人工智能有可能彻底改变医疗保健，因为它可以比人类更有效地指数级分析数据。虽然数据可能有助于其他部门锁定某些客户，或提供销售见解，但如果足够多的组织允许人工智能访问足够多的数据，人工智能实际上可能有助于拯救生命。

这可能包括分析各种规模的趋势，这可能有助于确定最佳的治疗方法，更准确的存活率，或风险评估。这可以帮助医生改善他们治疗病人的方式，也可以让病人对他们的健康状况有一个更准确的了解。人工智能还将帮助医疗保健公司大幅提高生产率。可以肯定的是，在 2020 年及以后，人工智能将在未来的医疗保健中发挥重要作用。

# **机器人技术**

我们都习惯了人类医生，但机器人无疑将在未来的医疗保健中发挥作用。当然，时间会告诉我们他们在文化上如何融入整个医疗保健文化。

无论如何，机器人已经在各种手术中提供帮助，并且在涉及到精度和控制问题时可以提供帮助。考虑到[达芬奇手术机器人](https://en.wikipedia.org/wiki/Da_Vinci_Surgical_System)早在 2000 年就获得了 FDA 的批准，这并不是什么新鲜事。这种设备安装在世界各地的国家，每台售价约 200 万美元。

当然，机器人技术在不断发展。Vicarious Surgical 最近宣布，FDA 已经批准了他们基于虚拟现实的机器人辅助设备。一家名为 NISI Limited 的初创公司正在[开发微型机器人](https://www.roboticsbusinessreview.com/content-from-our-sponsor/tiny-surgical-robots-require-equally-tiny-components/)，用于插入人体开口处。机器人将通过肛门或阴道插入。

该公司去年创造了历史，当时它宣布，由于微型机器人被插入直肠，它能够在 2018 年对活猪进行妇科手术。该公司计划在 2021 年前进行第一次人体手术。

# **可穿戴设备**

你很有可能熟悉可穿戴健康设备，无论是 Fitbit 还是 Apple Watch。迄今为止，已经售出超过 1 亿个 Fitbit 设备，它是一个很好的活动追踪器，可以确保个人得到足够的锻炼。

毫无疑问，这项技术在未来将会越来越多地融入到健康信息技术中。世界上最强大和最有影响力的科技公司之一谷歌[支付 21 亿美元](https://www.theverge.com/2019/11/1/20943318/google-fitbit-acquisition-fitness-tracker-announcement)收购 Fitbit 的事实证明了所收集的技术和数据。

这一趋势最近也获得了巨大的推动力。具体来说，埃森哲估计，美国消费者使用可穿戴设备[的比例从 2014 年的](https://www.businessinsider.com/wearable-technology-healthcare-medical-devices) 9%跃升至 2018 年的 33%。

这不仅仅是关于 Fitbit 设备——可穿戴设备还包括可穿戴监护仪，无论是涉及血压监护仪还是心电监护仪。例如，[最近推出的 Heartguide 允许用户](https://omronhealthcare.com/products/heartguide-wearable-blood-pressure-monitor-bp8000m/?utm_source=cj&utm_medium=affiliate&cjevent=115e276d181a11ea83c0022b0a1c0e10)随时随地测量血压。

# **结论**

有许多医疗保健 IT 趋势值得关注，并且不可能预测哪一个将定义未来。然而，很明显，在医疗 it 领域，上述趋势将是不可或缺的。事实是，在医疗保健方面，无论是可穿戴设备、应用程序还是虚拟医疗，用户都希望获得更便捷、更个性化的体验。

机器人和人工智能方面的技术进步也将在健康 it 方面发挥相当大的作用。收集的新数据对于降低成本、提高生产率和创造更好的患者体验至关重要。当然，这也意味着数据将不得不受到前所未有的保护，这就是为什么网络安全必须仍然是重中之重。
