你好，这几天我想了很多。我得出的结论是，写这个帖子很重要。你可能想知道为什么？

我被亚马逊拒绝了。这是人生最大的失望之一。这背后的原因是我是一个普通的开发人员，也是一个随着成长会忘记事情的人。问我的问题非常简单，比如“数组中插入和删除的复杂度是多少”或者“什么是堆栈和队列？”。我真的回答错了吗？他们永远不会告诉你拒绝的理由。
