让我们开始吧—

以下是在Python中使用地图、类、函数和参数时需要了解的一些关键概念:

  1. Map: Map是一个内置的Python函数，它以一个函数和一个iterable作为输入，返回一个新的iterable，函数应用于每个元素。Map可用于将函数应用于列表、元组或任何其他可迭代对象中的每个元素。
  2. 类 :类是面向对象编程中的一个基本概念。它们允许您用自己的属性和方法定义自定义对象，并封装和重用代码。类可用于模拟现实世界的对象和过程，使您的代码更加结构化和有组织。
  3. 函数: 函数是可重用的代码块，可以从程序的多个部分调用。函数可以接受参数和返回值，这使它们成为封装逻辑和提高代码重用的强大工具。
  4. 实参: 函数可以接受零个或多个实参，实参是函数被调用时传递给函数的值。Python中有几种类型的参数，包括必需参数、默认参数、关键字参数和可变长度参数。理解如何使用每种类型的参数是在Python中使用函数的一个重要方面。
  5. 方法装饰器: 方法装饰器是Python中一个强大的特性，允许你修改类中方法的行为。通过用一个函数包装一个方法，将装饰器应用于该方法，该函数将原始方法作为参数，并返回一个以所需方式运行的新函数。

1.地图

在Python中，map()函数将给定的函数应用于给定的可迭代结构(即列表、元组等)的每一项，并返回一个map对象。

语法—

映射(函数，可迭代)

实施1 —

numbers = (10, 20, 30)
result = map(lambda x: x+x, numbers)total = list(result)
print(total)

输出—

[20, 40, 60]

实施2 —

list_one = ['monday', 'tuesday'] 
  
t = list(map(list, list_one)) 
print(t)

输出—

[['m', 'o', 'n', 'd', 'a', 'y'], ['t', 'u', 'e', 's', 'd', 'a', 'y']]

3.功能

函数用来为你的代码提供更好的模块性、可重用性和可读性，它只不过是一个有组织的、可重用的代码块。要创建一个函数，你需要使用关键字def，如下图所示

实施1 —

def multiply_two_numbers(first_number,second_number):
    multiply_result= first_number * second_number
    return multiply_result#call the multiply function
multiply_two_numbers(10,8)

输出—

80

实施2 —

如果你想给一个函数传递多个参数或关键字参数，那么使用*args和**kwargs 。使用 *args —非关键字参数**kwargs —关键字参数

def addNum(*add_numbers):
    result=2+add_numbers[2]+add_numbers[6]
    return result
#call the function
addNum(5,3,19,46,66,26,10,34)

输出—

31
