# 首先你要养成习惯

…然后习惯塑造了你。

虽然有目标是好的，但有一个过程更好，朝着目标努力应该是你喜欢做的事情，无论是顺境还是逆境。

为了这篇文章，让我们把重点放在写作上。这门手艺有什么让你开心的地方？是你在社交媒体上得到的喜欢吗？无中生有的想法？因为你想变得富有和出名？

真的好好想想。有你的道理吗？很好。

现在，如果不能保证你能实现你的梦想，你还会选择写作吗？毕竟，你可以写一辈子书，但仍然贫穷、默默无闻、不受欢迎。

“人其实做事都没有成功的保证？”你可能会问。

是啊。很多。许多了不起的吉他手、游戏玩家、溜冰者、舞蹈家和程序员开始练习他们的技艺是为了热爱这个过程，而不是渴望成为第一。

> *为什么我们知道拉大提琴需要努力，但我们却把写作归功于灵感的魔力？*
> 
> *—安·帕切特*

亚历克斯·霍诺德今天因他的攀登技艺而闻名，只是因为他尽可能多的练习。但是许多人倾向于忽略他花了几十年来磨练他的手艺，以及他为了能住得离岩石更近而做出的牺牲。

那些年，他一直默默无闻，直到 17 年后才得到认可。他住在自己的货车里，除了每天爬，什么也不能做。你呢？你能不能减少琐碎的事情，这样你就可以专注于你的激情了？

你也可以成为写作界的亚历克斯·霍诺德。你只需要在这门手艺上投入足够的时间。对我们许多人来说，这意味着不可避免的——也是可怕的——坐下来写作。

如果你还没有这样做，但是你有通过写作创造美好事物的梦想，那么我想你最好从每天写一个句子开始。

# 从小处着手的好处

我们很快就忽略了我们为实现目标所付出的最少努力，以至于多年来我们什么也没做，只是回头看看，想知道我们的时间都去了哪里，为什么我们甚至没有迈出实现目标的第一步。

有这样一句话:“人们经常高估他们一天能做的事，低估他们一年能做的事。”当你认为自己优秀到不能一天只写*一句话的时候，这就是你正在做的事情。*

你认为你可以——也应该——在一天内写更多的东西，让事情变得有价值。但这往往会导致一年的不写作。我知道，因为这正是我近十年来一直在做的事情，我相信你们当中也有人做过同样的事情。

# 我自己的小步伐

一年前我采纳了这个建议，开始时也觉得很愚蠢。一句话在大局中有什么用？就是这样。一旦你放下一个句子，你就会忍不住去打另一个，一个又一个。你自己试试看。

我自己的例行公事很快变成了每天 250 个单词，在过去的六个月里一直如此。这是一个对我来说具有挑战性的目标，但即使在我最糟糕的日子里也很容易回到这个目标。通常我最终会写出 500 个单词，这还不包括我的小说、写作练习和写日记。

作为一个热衷于拖延的人，我很惊讶我发现现在很难*而不是*写作，自从我开始这个惯例以来，我从来没有跳过一天写作。

我学到的是，写作的恐惧不在于实际的工作，而在于开始的行动，这就是设定可实现的目标有巨大帮助的地方。因为持之以恒地写作会比紧张地写作让你走得更远，即使如此，正如老话所说，也要从一小步开始。

一小步。
