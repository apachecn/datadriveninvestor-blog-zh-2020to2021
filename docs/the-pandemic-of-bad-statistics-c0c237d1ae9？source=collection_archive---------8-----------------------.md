在某种程度上，我们被迫走向抽象思维:我们生活在一个不同于我们进化祖先所见的世界，主要是我们的概念思维能力试图跟上我们环境变化的速度，以及它转向哪种数字化、基于知识的领域。但这不是一项微不足道的任务。我们的大脑没有优化为理性和客观:[维基百科列出了大约 200 种认知偏见](https://en.wikipedia.org/wiki/List_of_cognitive_biases)，即我们的感知扭曲现实并阻止我们做出理性判断的心理模式。

## 数字的虚假自信

![](img/90c5e938aa8caeecfb92ad16cd8ad8bd.png)

我们所遇到的这个世界是不确定的。现代神经科学理论将我们的大脑视为在不确定性下不断尝试做出最佳决策的装置。

正如我们所知，当疫情在全球蔓延，威胁生命、工作和社会生活时，不确定性无处不在。

世界的抽象摆脱了现实的噪音和喧嚣，在面对不确定性时假装出一种客观性。数字给人一种牢不可破的印象，给我们一种稳定的感觉。他们让我们感到安慰，为什么不呢？数字提供了一个非常有用的工具，可以把秩序带入这个世界，并成功地操纵它。它们可能是自发现火以来人类取得的最重要的技术进步。

但数字并不总是等于数字。数字伴随着一种内在的风险:抽象是困难的，科学探究是困难的，数字可以隐藏它们起源背后的斗争，它们辐射的客观真理光环背后的起源的不确定性。

## 统计学的困难

[***统计***](https://en.wikipedia.org/wiki/Statistics) ，根据维基百科，关注的是 ***数据的收集、组织、分析、解释和呈现。***

数字是表示数据的主要方式之一。这些天，很多数字在媒体和公共话语中浮动:死亡率、病例总数、R0 因子、对策有效性的估计……但通常情况下，这些数字背后隐藏着没有答案的问题。

在假设什么数据告诉你现实的客观状态之前，必须回答一些关键问题:

1.  ***数据是如何收集和整理的？***
2.  ***是如何呈现的？***
3.  ***如何解读？***

## 数据收集的重要性

新冠肺炎对全球社会来说几乎是一个前所未有的挑战(别说气候变化了……)，让全世界的人都屏住了呼吸。因此，在这种环境下，寻找能让我们对实际发生的事情有一种确定感的数字是很自然的。

但是防止病毒传播的措施不是科学实验，所以我们应该非常小心地对待它。在几个方面，对病毒的测试与实际的实验有很大的不同，偏见也随之而来。记住这一点非常重要，事实上是这样的，这些数字是有保留的。

1.  谁会接受测试？大多数对来自“高风险地区”(如伊朗、意大利和中国)的人进行测试会导致选择群体偏差，从而导致分布偏斜，表明大多数来自高风险地区的人已经被感染，尽管来自其他地方的人也可能被感染，但未被检测到。
2.  比较国与国之间的数据价值有限，因为国与国之间的测试数量差异很大。虽然韩国在高峰期每天进行约 10000 次检测，德国也没有落后太多，但其他国家检测的次数少得多，因此检测到的感染数量也少得多。
3.  在某些地方，在某些时期，虽然确诊感染新冠肺炎病毒的患者数量呈指数增长，但检测数量也在迅速增加。原则上，这可能导致检测的大幅增长，即使感染人数保持不变。
4.  许多人几乎没有症状或只有非常轻微的症状，因此许多人不会不被发现，特别是如果测试能力过度紧张，因此仅限于来自选定群体的一小部分人。华盛顿的情况很好地强调了这一问题，据追踪，该病毒在第一个确诊病例前几周就出现了。

所以在我们用 ***解读*** 的数据之前(某某是死亡率，某某是感染病人的数量)，我们需要了解 ***的数据是如何被收集的*** 。

几天前，我的手机屏幕上出现了一个窗口小部件，上面用红色字体赫然显示着确诊病例数:201463 人感染了冠状病毒！考虑到全球病例的真实数量很容易相差 10-50 倍，我不认为假装把它们算到一个人身上有助于理解数据收集过程的难度。

病死率同样经常被抛出，但也有几乎同样多的不确定性:一个巨大的混杂因素包括人口统计数据(德国高达 70%的患者是从意大利滑雪旅行归来的健康年轻人，这导致了另一个大的选择群体偏见)，而在意大利，很大一部分受影响的人是老年人，部分原因是意大利的老年人更强烈地融入社会生活。此外，意大利可能还有更多未被发现的病例(考虑到 70 名从南蒂罗尔度假归来的德国人被检测为阳性，而当时整个国家只有 2 例确诊病例)。这一点，以及德国越来越早地开始测试的事实，已经导致两个表面上相对相似的国家之间的死亡率相差近 50 倍。

此外，还需要考虑感染和康复之间的时间差、重症护理的有效性、吸烟和空气污染的作用(在意大利和中国较高，在男性中更普遍)、该国的人口结构、医院的容量、哪些患者被计入新冠肺炎死亡人数的问题(第一个德国患者是一名接受姑息治疗的 78 岁晚期癌症患者，因此人们可以争论他的死亡在多大程度上应真正由新冠肺炎负责)等。

因此，说“死亡率是这样那样的”，“仅仅根据这些数字来判断新冠肺炎到底有多危险”是误导人的。如果我们谈论一个死亡率，我们需要知道它从哪里来，它到底在说什么。

## 采用贝叶斯框架

在贝叶斯统计中， ***概率表达了我们对一个事件*** 的相信程度。对一个量的贝叶斯估计总是包含我们认为我们对该量的了解，加上我们对该量固有不确定性的估计。

![](img/b011cd1c16f60ceea3c4ca4ed57d6b15.png)

A set of Gaussian probability distributions with different covariances. Source: Inductiveload / Public domain

数字表达了我们对这个世界的认识:但由于这种认识必然是概率性的，因此贝叶斯统计中的数量由概率分布(可以是上图中的钟形曲线)而不是单个数字来表示。分布的宽度代表了我们估计的确定程度。图表的最高点是我们的最佳猜测(高斯分布的平均值)，但是如果分布真的很宽，我们的最佳猜测并不能告诉我们太多。

正如这篇[对我们可能采取的应对措施的深入分析](https://medium.com/@tomaspueyo/coronavirus-the-hammer-and-the-dance-be9337092b56)详细解释的那样，在新冠肺炎问题上有很多未知因素，有太多未知因素让我们无法过于自信地运用任何数字(这也解释了为什么强有力的措施是我们目前的最佳政策，因为它们为我们赢得了时间来获得更清晰的画面)。

![](img/61635a778783b8a9d9075795485fb24a.png)

这张图表已经环游世界，它来自伦敦帝国理工学院尼尔·弗格森等人周末发表的一篇论文。

不管其信息有多重要(它导致了美国和英国的政策变化)，图表显示曲线的方式具有误导性。模拟中插入的隐含参数是什么，它们的置信区间有多大？天气/不同的社会距离措施/社会结构/新兴疗法的影响都是不确定的，这些因素都没有通过实证研究确定，但迄今为止都是猜测。

正如杰瑞米·霍华德在他对新冠肺炎情况的实用总结中所说的，虽然这些曲线看起来很可怕，但它们周围的误差线可能接近曲线本身的大小。

## 承受不确定性

一句话:面对不确定性，保持冷静可能很难，但这是有一定智慧的。

不幸的是，在政客中，承认不确定性往往被解读为软弱的表现。这就是为什么我认为科学界有责任强调它在评估正在发生的事情中发挥的作用，这对我们应该采取的措施意味着什么，以及为什么这种不确定性是我们需要更多时间通过对病毒进行更严格、更科学的评估来慢慢征服它，然后决定最佳长期战略的最佳原因之一。

当疫情的乌云笼罩在我们头上时，我们喜欢拿着数字不放。但是，在更清楚的事实出现之前，在全球社会对局势有更坚定的把握之前，最好是承受不确定性，然后为了自欺欺人地安慰我们而使事实永久化，或者在另一个极端，使我们陷入恐慌，这种恐慌来自于认为我们比我们实际上更了解正在发生的事情。
