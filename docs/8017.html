Python lambda函数

Python lambda函数，也称为匿名函数，是没有名称的内联函数。它们是用关键字lambda创建的。

Python lambda函数仅限于单个表达式。它们可以在任何可以使用正常功能的地方使用。

在这个应用程序中，它用于在按下按钮时运行函数调用。
