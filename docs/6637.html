1.λ表达式

Lambda用于使用“lambda”关键字创建小型匿名函数，可以在任何需要函数对象的地方使用。它可以有任意数量的参数，但只能有一个表达式。

在几秒钟内读取和处理大型数据集—第1部分

在几秒钟内处理十亿行..

语法:

λ自变量 : 表达式

实施—

#lambda arguments : expression
#A lambda function that adds 10 to the number passed in as an argument, and print the result
x = lambda a, b, c : a * b + c
print(x(5, 6, 8))

输出—

38

它可以在另一个函数中使用。

实施—

def cal(n):
    return lambda t : t ** nvar = cal(3)print(var(12))

输出—

1728

2.函数参数

如果你想给一个函数传递多个参数或关键字参数，那么使用*args和**kwargs 。

使用的符号-

* args非关键字参数

**kwargs —关键字参数

实现(*args —非关键字参数)—

def addNum(*add_numbers):
    result=2+add_numbers[2]+add_numbers[6]
    return result#call the function
addNum(5,3,19,46,66,26,10)

输出—

31

实现(**kwargs —关键字参数)—

def my_function(**legend):
    print("His last name is " + legend["lname"])my_function(fname = "Steve", lname = "Jobs")

输出—

His last name is Jobs

3.函数注释

Python 3.0增加了函数注释，这在类型检查时很有帮助，因为Python支持动态类型。它们收集关于使用的参数类型和函数返回类型的信息。它们可以和单个参数、函数参数、嵌套函数、装饰符等一起使用。

实施—

def cal(x:int, y: float, z: int) -> float: print(x * y + z)cal(5,6.5,7)

输出—

39.5

4.有序直接

OrderedDict 是高性能容器数据类型之一，是Dict对象的子类。它保持键的插入顺序。在删除或重新插入密钥的情况下，顺序保持不变。

实施—

print ('Ordered Dict')
dict2 = collections.OrderedDict()
dict2['a'] = 'A'
dict2['b'] = 'B'
dict2['c'] = 'C'for key, value in dict2.items():
    print (key, value)

输出—

Ordered Dict
a A
b B
c C
