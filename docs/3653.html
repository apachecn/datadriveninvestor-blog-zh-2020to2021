让我从房间里的大象开始。亿万富翁和百万富翁应该被课以重税，他们应该停止躲在他们的慈善声明后面。

现在，和我就财政政策进行过辩论的人会说，“普拉文，是你吗？”。当我声称我不希望政府陷入赤字时，我把自己定位为一个反对政府超支的人。这并不意味着我不希望政府资助全民医疗保健、免学费教育、政府资助的基础设施、全民基本收入等。这意味着政府应该投资于这些基本需求，但应该对富人征收足够的税，这样政府就不必出现赤字。是的，就像伯尼·桑德斯和杰里米·科尔宾提出的那种政策。

我认为，政府应该提供全民医疗保健、免费高等教育和公共交通，而不是私营部门。然而，我认为，其他行业，如电信，应由私营企业在强有力的法规和对个人的高税收下经营。在流行文化中，人们会称我为社会主义者，而经济学家会称我为社会民主党人。

这篇文章更像是一堂历史课，讲述了几位总统和总理(实际上是两位)如何塑造了全球经济，加剧了困扰当今全球经济的财富不平等。这几个领导人是伯尼·桑德斯、杰里米·科尔宾、保罗·克鲁格曼和中右翼政客自称为社会主义者以区别于极右翼的原因，而这些人实际上是社会民主党人。

现在，在我进入历史课之前，我们需要了解美国、英国、日本和斯堪的纳维亚等几个经济体的公共政策是如何影响其他国家的公共政策的。这些经济体颁布的公共政策通常构成大多数其他国家，特别是发展中国家公共政策的基础。更确切地说，美国和斯堪的纳维亚通过的公共政策构成了其他国家制定的大多数公共政策的基础，甚至在我居住的国家印度也是如此。这就是为什么美国国会和英国议会被认为是“世界上最有影响力的审议机构”。

理解前一段很重要，因为通过跟踪这些经济体的公共政策，我们可以了解像我国这样的国家更有可能在这里起草什么政策。上完这堂简短的历史课，你会更好地理解它。

好吧，我们开始吧。

如今，世界各地的右翼政党认为，减少税收和加强去监管将改善经济，因为这样的政策将增加投资和促进增长。经济学家将这种意识形态称为涓滴经济学，在美洲也普遍被称为“里根经济学”。涓滴经济学认为，如果你减少对富人的税收和监管，那么减少的税收将把更多的钱放在富人手中，他们将把这些钱再投资到经济中，作为刺激经济的一种手段。通过再投资，人们相信富人可以创造更多的生意，从而创造更多的就业机会。

在20世纪80年代之前，即使在当时的极右翼和自由主义政党中，涓滴经济学也并不流行。就连极右翼政党也认为，对富人实施适度的税收和监管是正确之举。过一会儿我再回到西方国家80年代以前的政策。

如果没有两个人，涓滴经济学就不会存在于今天的右翼政党中。在20世纪80年代，有两位非常受欢迎的西方领导人——美国总统罗纳德·里根和英国首相玛格丽特·撒切尔。在此期间，西方国家面临着通货膨胀危机，因此西方经济体试图通过投资来刺激经济。有很多方法可以将现金注入公众手中，以增加需求，重振经济。

罗纳德·里根和玛格丽特·撒切尔是私营企业和自由主义观点的大力支持者。也有非常有魅力的人。他们以绝对优势赢得了选举。因此，几乎他们提出的任何政策都没有遭到多少反对就通过了他们的立法机构。

因此，为了避免陷入衰退，这两位领导人诉诸于取消对投资的监管，尤其是对投资银行的监管。他们的想法奏效了。但是也有很多副作用，我们稍后会讨论。

这两位领导人从取消投资管制开始，然后慢慢开始缩减政府在教育、能源、环境等方面的支出。大型公司喜欢这些想法。在罗纳德·里根之前，在上世纪五六十年代，亿万富翁的最高边际税率为90%。现在，很多税收都被取消了，很多公司根本不用缴税。这使得这些公司可以将一部分多余的钱再投资回去，而其余的钱则被用来充实自己。

这些政策取得了巨大的短期成功。然而，这两个国家的赤字增加了，但这在短期内不是什么大事。西方国家得以避免衰退，通货膨胀稳定下来，就业机会不断增加。这推动了罗纳德·里根和玛格丽特·撒切尔的政策在全球舞台上的流行。

他们受欢迎的另一个原因是，减少的法规允许公司在其他国家投资更多，这增加了这些国家的就业，特别是在非洲和亚洲的许多国家。世界各国政府都相信涓滴经济学。许多国家开始转向这种放松管制和减税政策。罗纳德·里根和玛格丽特·撒切尔被认为是各自国家的最佳领导人。

由于这些政策的流行，这些国家的未来领导人开始了放松管制的文化。通过解除对公司的环境限制和最低工资限制，他们相信公司将能够提供更多的就业机会，从而改善中产阶级及其支出。

但是，在2008年金融危机之前，很多事情都没有引起人们的注意。一些国家短暂地遵循了涓滴经济学，然后划定了解除管制和减税的界限。加拿大、德国、韩国、新西兰是这些国家的一些例子。放松管制的流行随着一些西方国家如美国、英国、巴西、墨西哥、爱尔兰等而结束。放松对两大行业的管制:银行业和能源业。

随着比尔·克林顿总统废除格拉斯-斯蒂格尔法案，银行业放松管制的高潮达到了。许多国家也采取了类似的放松管制措施。放松管制使得银行在提供贷款、投资等时不受约束。最终导致了2008年的全球金融危机。

想象一下世界各地银行系统放松管制的缓慢过程，就像一座积木塔。通过拆除塔的每一块(解除管制)，塔变得越来越不稳定。最后，当你再拉一片，塔就会倒塌。

放松对银行系统和能源部门的管制始于美国和英国，由于短期利益，世界各地都采取了类似的政策。2008年，我们面临着银行体系的崩溃，只是在美国发生这种情况后，其他国家才意识到错误，并制定了强有力的监管措施。如果我们没有重新引入监管，那么太多国家将面临同样的危机。

大约在同一时间，许多公司对能源部门的放松管制允许他们钻探和燃烧大量的石油，尽管科学家对全球变暖发出了警告，但这些放松管制允许能源公司向大气中释放大量的碳足迹。这加速了全球变暖，以至于现在我们正面临着一场生存危机，科学家们预测，我们这一代人将是第一代会因为全球变暖而大量死亡的人。

现在，你能理解美国和英国制定的政策的重要性吗？罗纳德·里根和玛格丽特·撒切尔在世界范围内发起了一场运动，为大公司和富人减税，并解除对这些公司的管制。他们的计划在短期内奏效，但这种放松监管的文化导致了我们现在面临的两个最大的危机。

现在让我们看看由于放松管制和对个人财富积累的宽容而面临的其他问题。

高税率抑制了亿万富翁。在美国，在罗纳德·里根之前，如果亿万富翁决定保留他们的财富而不是再投资于经济，他们的边际税率为91%。所以，如果我有10亿美元，我决定保留它，我必须把91%的钱交给政府。如果我决定将这笔钱再投资于一家公司或慈善事业，那么他们就不需要缴纳91%的税款。这种税收迫使企业向自己的公司投资，随之而来的是，员工的工资提高了，中产阶级也壮大了。

但是在里根和撒切尔的政策之后，由于非常低的税收，富人可以很容易地积累财富。因此，个人专注于积累过多的财富，最终导致了我们今天面临的收入和财富差距。世界上最富有的1%人口拥有全球44%的家庭财富。如果没有罗纳德·里根和玛格丽特·撒切尔推行的减税政策，这是不可能的。

这些减税的想法仍然在世界各地流行。小企业和正常个人从减税中受益匪浅。他们倾向于以一种有用的方式投资这些钱，比如改善他们的业务、保险、教育或个人基础设施需求。由于减税，大企业和富人倾向于积累财富。

但是仍然存在这样一种误解，即对富人和公司减税将有助于刺激经济。并没有。这只会增加他们的个人财富。然而，为小企业减税被证明是非常有用的。在印度，最近，政府给大公司减税，但没有给小企业减税。他们希望减税会增加投资，但这并没有发生。如果他们给小企业减税，那么中产阶级就会增加。

此外，由于涓滴效应的经济政策，公司可以向东方国家转移，如亚洲和非洲，以满足他们对体力劳动的需求。这给东方国家带来了大量低技能工作，这是一件好事。但是，这些公司的许多人失去了工作，因为这些工作现在由东方更廉价的劳动力完成了。这是世界各地民族主义政党和反全球化情绪兴起的原因。作为这种情绪的副产品，这些民族主义政党诉诸族裔民族主义和种族主义来推行其反全球化的意识形态。

现在，让我回到为什么我说，“亿万富翁和百万富翁应该被课以重税，他们应该停止躲在他们的慈善声明后面。”。

富人通过慈善事业贡献了很多。但这只是其中的一小部分。由于公司的贪婪，这些富人每年逃税超过1500亿美元。他们这样做是因为罗纳德·里根经济政策的一个重大缺陷。20世纪80年代以前，税收政策相当简单。如果你积累了太多的个人财富，你要付很重的税，如果你把它们再投资到公有企业，你要付的税就会少一些。

罗纳德·里根之后，税收系统变得如此复杂，导致了太多的漏洞。在政策制定中，有一个简单的规则。政策越简单，漏洞就越少。罗纳德·里根之后的人，以奇怪的方式修改了税收制度。许多国家也加入了这一潮流，并实施了类似的税收制度。印度的结构和漏洞与美国非常相似。

其中一个漏洞是，你可以在一个税率很低的国家注册公司，而在另一个国家开展所有业务。一些奇怪的漏洞让你可以在私人岛屿上注册公司，避免很多税收。在上世纪80年代之前，没有一个国家有这样的漏洞。

这些富人有如此多的钱，以至于他们已经花了巨额资金投资于研究，只是确保人们相信20世纪80年代的放松管制策略可以帮助人们。

让我举一个这样的例子。

卡托研究所是一个智囊团，由科赫工业的所有者科赫兄弟创立和资助。科赫兄弟的总净值为505亿美元。卡托研究所的唯一动机是进行宣传，并专门资助促进涓滴经济学的研究。卡托研究所在全世界都有影响力，他们已经在许多欧洲和西方国家进行了宣传。通过影响这些国家，他们有效地游说其他市场，如亚洲市场。

卡托研究所的工作是公开的，我不想在这里推动一个议程。许多研究人员已经清楚地表明，卡托研究所发表的研究是基于选择性的数据收集和“断章取义”的比较。

但这是少数人拥有大多数财富的问题。他们可以很容易地建立这样的组织，并很容易地资助政治家的选举。公司资助政客是全球众所周知的罪恶。

像科赫兄弟这样的人隐藏的伪装是慈善事业。我请求你要么等一篇文章，要么读一读阿南德·吉里达拉达斯的杰出著作《赢家通吃:改变世界的精英猜谜游戏》。

那么，我们如何解决这个问题呢？这里有解决办法，税，税，税。

我不反对人们挣钱。我反对一小部分人拥有全世界一半的家庭财富。就这么简单。

但是，在之前的一篇文章中，我确实提到在20世纪70年代和80年代，斯堪的纳维亚国家70%的财富税不起作用。是真的。问题是，在斯堪的纳维亚国家，他们对一个人的全部财富征税。例如，如果我以80，000美元购买了一家公司的一些股票，那么我将因拥有这些股票而被征税。但拥有股份只是一种资产。相比之下，你每年因拥有一块土地而根据土地的价值被征税。在某一点上，你必须卖掉那块土地来交税。那个想法是混乱的。

此外，这些国家的财富税也适用于不太富裕的人。这不是个好主意。伊丽莎白·沃伦和杨安泽等人提出的计划专门针对经济中最富有的5%的人。我们存在巨大的不平等，对这些人征税足以使社会保障计划的预算增加一倍。在印度，像这样的财富税将允许我们为全国40%的人支付普遍的基本收入，这是令人惊讶的。

许多经济学家和政策制定者现在已经提出了重大修改。我们可以通过分割资本收益并单独对其征收重税来解决许多此类问题。

现在，如果你读过一些关于经济学的文章，你会读到，“向富人征税将扼杀增长”。我只会对那些人说一句话，“你们错了”。

像凯托研究所这样的智囊团已经给我们洗脑了，他们认为高税收意味着低增长。这是错误的。对小企业征收更高的税意味着更慢的增长。对百万富翁和亿万富翁的个人财富征收高额税收与经济增长并不相关。

而且，这种高税收的想法在20世纪50年代和70年代一直有效。许多经济学家称那个时期为资本主义的黄金时代。中产阶级以惊人的速度增长，贫困以前所未有的速度减少，财富不平等减少等等。如果我们想恢复那样的增长水平，那么只有恢复那些经济政策才是正确的。

在20世纪50年代，我们对富人的个人财富进行了大量征税，这削弱了他们对经济的所有权，这些钱可以用于资助各种社会福利计划。事实上，美国是涓滴经济学的旗手，对91%的个人财富征收最高的边际税率。

记住，对小企业征税只会扼杀增长。但对个人财富征税与此无关。事实上，正是保守派总统德怀特·D·艾森豪威尔想要高税收制度，并认为通过对个人财富征收高税收，我们迫使富人重新投资于经济，这样他们就可以避税。依我看，这是个绝妙的主意。

此外，向富人征税并不意味着我们需要一个强大的政府。我反对让政府拥有太多权力的想法。我们可以淡化权力结构，尽管允许政府获得巨额资金。我们这样做的方式是增加福利国家的巨额资金，减少对军队和执法部门的拨款。嗯，有很多证据表明，如果有一个适当的福利国家，犯罪率就会下降，斯堪的纳维亚国家就是证明。

所以是的，就是这样。从20世纪50年代到70年代，向富人征税一直有效。它在世界各地都有效。如果没有两位领导人，我们就不会有今天的财富不平等。事实上，如果罗纳德·里根是某个欧洲或亚洲国家的总统，没有人会认真对待他的政策。但他是世界上最富有国家的领导人，这是其他国家追随他的唯一原因。这就是为什么我说理解这些国家的政策很重要，因为它们最终也会影响其他经济体。
