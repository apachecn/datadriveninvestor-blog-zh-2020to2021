图论

图 是一组顶点和一组连接一些顶点的边。我们将考虑简单、无向、连通图:

  • 如果顶点之间没有回路或多条边，则一个图是简单的
  • 如果边没有方向，则图是 无向的
  • 一个图是 连通的 如果图中的每个顶点都通过一条路与其他顶点相连

我们可以把一个图想象成一组顶点和边，并通过观察它来回答关于这个图的问题。然而，对于诸如 社交网络图 之类的大型图，这变得困难得多。相反，我们从图中构造矩阵，如 邻接矩阵 和 拉普拉斯矩阵 并研究它们的性质。

谱图论 是研究邻接矩阵(以及其他关联矩阵)的特征值与g的结构的关系。

邻接矩阵

具有n个顶点的图G的 邻接矩阵 AG是大小为n的方阵，使得如果顶点I和j由边连接，则 Ai，j=1 ，否则 Ai，j=0。

我们可以使用networkx来创建图g的邻接矩阵。函数nx.adjacency_matrix返回一个 稀疏矩阵 ，我们使用todense方法将它转换成一个常规的Numpy数组。

例如，画出有5个顶点的 完全图 并计算邻接矩阵:

G = nx.complete_graph(5)
nx.draw(G,with_labels=True)Out:

让我们编写一个名为triangles的函数，它采用一个方阵M并返回总和—

其中 λi 是对称矩阵的特征值

注意 M=A 如果M是对称的。如果输入M是邻接矩阵，返回值是图G 中三角形的数量。

def triangles(M):
    A = (M + M.T)/2
    eigvals, eigvecs = la.eig(A)
    eigvals = eigvals.real
    return int(np.round(np.sum(eigvals**3)/6,0))
