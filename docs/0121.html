X-Order是一家投资和研究机构，致力于研究开放金融领域的价值获取。我们努力成为新金融和与科学和研究相关的跨学科领域之间的桥梁。它的创始人是托尼·陶，他也是NGC风险投资公司的合伙人。

我们采访了Dimension的创始人严苏吉和首席技术官。他们对技术的理解和对梦想的执着给我们留下了深刻的印象。

嘉宾访谈

维度简介

Dimension成立于2018年，致力于构建新的用户数据系统，为下一代互联网应用铺平道路。

Maskbook 是Dimension为保护用户隐私而开发的免费开源扩展。它适用于Chrome、Edge、Brave(所有基于Chrome的浏览器)、Firefox。iOS和Android版本也已经开始公开测试。

Maskbook允许用户在脸书上发布加密帖子。只有用户和他们的朋友可以解密内容。简而言之，只有得到你许可的用户才能查看你发布的内容，即使是脸书和Maskbook的开发者也无法查看。

你能介绍一下Dimension是做什么的吗？这个项目进展如何？

刘易斯

Dimension的愿景是，在未来，当隐私互联网的架构逐渐成型时，我们可以在其中发挥重要作用。同时，在我们拥有足够多用户的情况下，我们可以帮助他们在互联网架构的隐私下更好地使用他们的数据。

由于我个人崇尚技术，我相信在我们的日常生活中，技术赋予我们的好处不仅限于金钱。我希望通过更好地利用我们的数据来改善我们的服务。

现在很多公司很好的利用了我们的数据，但是没有站在我们的角度思考问题。他们只是简单的商业化——将我们的数据货币化。

然而，我相信从用户的角度来看，一些数据处理可以给我们带来更多的力量。如果我们能够更加定制化地处理数据，甚至在智能机器人的协助下，它可以在生活的各个方面帮助我们。

这是长远的眼光。但在此之前，我们需要帮助用户构建底层基础设施。

“我们至少应该先帮他们拿回数据的所有权——重新获得数据的私有化。”

苏姬

我们想在Web 2.0和Web 3.0之间架起一座桥梁。并不是完全背离Web 2.0，而是更好的弥合双方的价值。我不认为Web 2.0会长期消亡，但是会出现无数的问题。我们希望在用户不离开互联网巨头平台的情况下解决这些问题。

你认为项目的哪个部分最具挑战性，你是如何解决的？

刘易斯

从技术角度来说，Dimension是一个应用密码学项目，最大的挑战也来自于应用密码学:基本上研究论文上提出的方法99%都不能应用。

我们需要整合许多传统的加密方法。由于技术和效率因素，许多由密码学设计的算法不太可能被计算机运行。把一些算法实现成代码可能不是很难，但是实现起来没有漏洞就很难了。这就需要很多人去学习，去落实。

另一个挑战是我们正在一个分散的系统中开发项目。如果是一个集中式系统，只需要在几个服务器上部署所有服务就足够了。然而，如果我们需要在一个分散的网络中运行我们的整个系统，这将产生许多新的和不可想象的问题。

去中心化身份是我们要开发的产品之一。我们希望当用户使用我们的一些服务时，他们不需要与中央服务器或中央服务提供商打交道。

“正是自由的基础赋予了用户身份和隐私。”

在去中心化的网络下实现这个功能有一个难点，就是先实现离线。我们需要确保您的网络或服务能够在没有互联网连接的情况下运行。我们现在用一个分散的数据库结构GUN DB来解决这个问题。然而，在这个过程中，我们仍然遇到了很多问题。例如，我们没有办法跟踪分散网络中操作和传输的一些数据，这导致对我们用户的了解有限。

此外，在分散网络中，多点同步也是一个难点。例如，如果你有两部手机和三台电脑，在极端的网络情况下，你很难将没有连接到互联网的设备的状态与其他设备同步。

苏姬

我认为除了技术之外最大的困难是社会视角的变化。视角的转换不是很直观，但是如果从社会科学的角度来看就很直观了。

我们正在做的事情类似于社会科学领域的传统工会。在互联网世界里，脸书就像一个巨大的工厂，有22亿人在里面工作。但是，这个工厂和18、19世纪的工厂不一样；它已经成为一个虚拟的东西。没有主管在背后鞭笞你，你也不知道自己在生产什么。然而，这种生产率关系与18世纪和19世纪的工厂是一样的，只是更加难以捉摸。

今天，当我们在互联网上与人交流时，我们需要在互联网开放市场上找到一种服务。这是因为大多数人不会在家里建立一个服务器或制作一个聊天软件，只是为了通过互联网与他人交流。

“这就好比你在互联网上的‘土地’被资本家拿走，从而被迫成为互联网上的‘无产阶级’。”

我们每个人在这个虚拟工厂里生产什么？很简单，就是数据。每个人都被自己的社交关系所困，进入脸书和推特等社交媒体。这就像18世纪的流氓从农村搬到城市，却被困在工厂里一样。

现在，我们生产数据组件和零件，这在政治经济学中是非常明确的。在过去，罢工和无产阶级革命是众所周知的。然而，如果你说像脸书这样的社交媒体软件在剥削你，这是不寻常的。

“在我看来，这就是互联网的价值转移。”

我们被一群虚拟工厂中的互联网公司包围着，他们让人们工作，却没有办法组织工人运动来争取每个人的权利和利益。

所以我们想做一件事情，能够把工人和用户的所有数据连接起来，让他们不仅知道自己被剥削还参与到适度的罢工中，赚到钱。

我们的一个产品Maskbook鼓励用户在形成一个免费的私人数据市场之前，在脸书上获得他们数据的所有权。

要实现这个目标，第一步就是加密用户的数据。除了加密，我们甚至可以发行稳定的加密货币，cryptokitties等。在原来的平台上。最后，编写我们自己的数据，类似于合同。这些计划正在逐步推进。

你为什么要发起维度项目？

刘易斯

我曾经做过 NLP 相关的研究，主要的数据来源是社交媒体。我需要抓取大量的非结构化数据，然后通过模型或信息提取直接处理它们——在处理它们之前结构化原始数据。

这些作品非常复杂，我已经做了很长时间了。然而，我后来意识到，在这漫长的时期里，我们从未质疑过我们所做的是否正确。这其实是一个哲学问题。

“我相信我做得很好，因为这个项目的目的是帮助人们，但它是一把双刃剑，因为同样的行为也可能侵犯人们的隐私。”

我们所做的所有分析实际上都是片面的，因为我们的数据太少了。最大的问题是，我们只知道用户在某个平台上的行为，因此对这个人的了解是不全面的。我们无法知道他到底是什么样的人。

因此，在一开始，我们希望从用户出发— 获得更全面、更全面的数据，其中一些数据他们可能知道也可能不知道，然后根据这些数据进行个性化推荐，给他们带来一些好处。

然而，这是不安全的，因为它侵犯了用户的权益。所以我们想先帮用户解决数据归属的问题，然后再帮他们做我之前说的事情。

2017年以来市场发生了很大的变化，你对行业有什么新的看法吗？

苏姬

最直接的感受就是融资变得更难了。2018年初以来，ico很多。市场上不好的加密货币正在驱逐好的加密货币，最终导致许多风险基金缺钱和运营不善。

即使仍有资金，投资后的撤出期也变得很短，有些甚至在一年后要求撤出。一般情况下，互联网基金的退出周期都是3+1，或者说3+2，时间不是很长，但至少给了初创企业更多的时间。

由于我们国家支持区块链，这导致一些以前不敢碰区块链项目的基金来找我们聊天。

你对财团区块链和公有区块链有什么看法？

苏姬

在我看来，有点像谷歌出来之前的硅谷。当时火热的创新项目是一个行业搜索引擎。马花藤还在《CFido时代》上写了一篇文章，讨论他从事的一个行业搜索引擎外包项目。这在当时是非常有利可图的生意。

美国的邮政、电信和水利局、图书馆、银行和保险公司都需要行业搜索引擎。就像现在法学院的一些搜索引擎一样，它能够搜索论文、政府报告等。，这在当时是一个热门的业务。许多硅谷投资者投资了B2B和B2G公司。但是Google出来之后，这些行业的搜索引擎公司都倒闭了。

在过去，搜索是一种特权，必须由图书管理员来完成。如果每个人都能使用搜索引擎，这种特权就会被侵蚀掉。这种垂直细分的行业搜索引擎只能在小行业生存。

我认为区块链工业也是如此。现在的问题是B2C的解决方案太难了，再加上大量的炒作泡沫，很多项目都倒闭了。因此，很多区块链公司会拥抱政府或企业，这至少能让他们赚点钱养活自己。

然而，开发一个纯B2B或B2G应用程序并不好。

“只要一个应用能整合所有这些功能，那么这些细分领域的应用就会失去竞争力。”

我们认为，在短期内，我们可以做区块链财团，但我们仍然需要努力成为一个整合所有功能的部分。这能让我们在未来生存下去。

从您的微信时刻，我们了解到Dimension的产品之一——mask book被脸书禁售。你打算怎么处理这件事？

苏姬

澄清一下，被屏蔽的是包含Maskbook.io或脸书网站Maskbook.io链接的信息。第一个版本(2019年7月推出)使用Maskbook插件加密信息，产生输出“用Maskbook.com解密这个”。需要加密的是这条语句之后的内容。

这是“增加黑客数量”和“病毒传播”的常用手段。Maskbook早期的核心用户有很多手段可以推荐自己的好友或者发布加密帖子。脸书不希望个人或群体经常发送许多链接，因为这将违反所谓的脸书社区标准。所以在2019年8月，屏蔽了这些链接。之后被屏蔽的用户可以申诉，我们也填了。然而，脸书的“社会标准”实际上是双重标准。虽然它在互联网上公开运作，但他们作为私人公司没有义务给你任何答复(至少美国法院会处理你)。

此外，该禁令实际上是针对脸书的所有服务，其中脸书、Messenger和Instagram被完全屏蔽。由于加密，WhatsApp无法被屏蔽，然而，脸书早就想解除加密。这也是当时WhatsApp创始人离开并反对脸书的原因。这些app占据了大部分欧美移动社交媒体流量入口。我们一上线就得到这种待遇，太震惊了。

但这种禁令只能屏蔽链接等集中项，而不能屏蔽密文本身(因为密文加密甚至可以做成表情符号、贴纸、图片等。).它也无法阻止其他通信方式，如发布Chrome网络商店链接、官方社区链接(如Twitter)。我们也在整合IPF来托管内容。最终，除非脸书封锁所有IPFs节点，否则他们无法真正封锁我们。

最近，在以太坊的《敌无双》本身期间，MetaMask公布了它的新插件系统。它的创始人丹也非常支持我们。他在MetaMask的 Github 上开了一期，讨论集成Maskbook的技术问题。如果整合完成，脸书将不得不在禁止我们之前杀死以太坊。

P.s:除了Maskbook，Dimension还推出了另一款产品Tessercube，可以让用户在微信等各种平台分发去中心化的“红包”。
