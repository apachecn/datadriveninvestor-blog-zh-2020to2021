尽管 63%的美国公众表示政府应该为所有人提供医疗保健，全民覆盖在美国仍然是一个非常棘手的考验，但是遍布六大洲的67个国家(其中许多是发展中国家)已经解决了医疗保健之谜。英国、欧盟、澳大利亚、新西兰和加拿大的既定官僚体系通过直接向银行账户存款来加速和简化财政援助的发放，而美国人则等待支票的邮寄。没有一项经济刺激措施能充分满足所有社会成员的需求，但没有一项措施像美国这样冷酷无情，至少人们不关心医疗保健，也不知道如何支付账单。虽然对赤字的担忧限制了美国，但新西兰财政部长宣布，该国将增加债务，以减轻新西兰人的金融痛苦；什么概念！

  1. 民治政府；一种政府形式，在自由选举制度下，最高权力属于人民并由他们或他们选出的代理人直接行使。
  2. 有这种政府形式的国家:美国和加拿大是民主国家。
  3. 权利和特权形式上平等的社会状态。
  4. 政治或社会平等； 民主 精神。
  5. 区别于任何特权阶级的普通人；普通人的政治权力。
