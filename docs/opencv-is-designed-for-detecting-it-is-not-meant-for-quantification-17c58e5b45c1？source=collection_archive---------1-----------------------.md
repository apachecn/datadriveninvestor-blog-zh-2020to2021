# 张量图像

图像是像素。这些像素具有与每个像素相关联的任意数量的值。

每个像素只有一个值的图像是标量图像。

多个值可以在一维或二维中排列，如矢量图像或矩阵图像。

彩色图像是矢量图像的一个例子，例如在 RGB 颜色空间中，每个像素的矢量有 3 个值，它是 3D 矢量。

向量和矩阵的推广是张量。秩 0 张量是标量，秩 1 张量是向量，秩 2 张量是矩阵。
