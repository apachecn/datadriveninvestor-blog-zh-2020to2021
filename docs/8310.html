需要了解的4个电子商务ETF

网上购物已经成为一种趋势

电子商务是零售业中增长最快的领域之一。根据精心研究的电子商务市场-全球机会分析和行业预测(2018–2025)，预计到2025年，全球电子商务市场将以11.1%的复合年增长率( CAGR )增长，价值24.3万亿美元。

随着电子商务的兴起，实体零售店面临压力。人们希望能够在自己舒适的家中购买他们想要的物品。以下4只交易所交易基金将帮助你了解引领电子商务发展的公司的最新动态:

1.全球X电子商务ETF ( EBIZ )

2.ProShares在线零售ETF ( ONLN )

3.放大网上零售ETF ( IBUY )

4.新兴市场互联网和电子商务ETF ( EMQQ )

1。全球X电子商务ETF(EBIZ)

EBIZ是一种ETF，旨在提供与 Solactive电子商务指数相对应的投资结果，该指数旨在为交易所上市公司提供风险敞口，这些公司有望从电子商务作为分销模式的日益普及中受益。EBIZ内的公司有拼多多、自由市场、 Etsy 、 Expedia集团。

截至2021年1月6日，EBIZ价格为31.97美元，费用率为0.50%。自2018年11月27日成立以来，EBIZ提供了约110%的总回报，年均总回报约为42%。

2。ProShares网上零售ETF(ONLN)

ONLN是一只ETF，寻求跟踪 ProShares在线零售指数表现的投资结果，该指数旨在衡量主要在线销售或通过其他非商店销售渠道销售的上市公司的表现。ONLN中的公司有亚马逊、阿里巴巴、 GrubHub 和 Qurate Retail 。

截至2021年1月6日，ONLN的价格为76.35美元，费用率为0.58%。自2018年7月13日成立以来，ONLN提供了约132%的总回报，年均总回报约46%。

3。放大网上零售ETF(IBUY)

IBUY是一只ETF，寻求与 EQM在线零售指数相对应的投资结果，该指数旨在衡量在线零售收入可观的上市公司的全球股票证券的表现。IBUY的公司有 Qurate Retail 、 Lands' End 、 Groupon 和猫途鹰。

截至2021年1月6日，IBUY的价格为116.44美元，费用率为0.65%。自2016年4月19日成立以来，IBUY提供了约363%的总回报，年均总回报约38%。

4。新兴市场互联网&电子商务ETF(EMQQ)

EMQQ是一只ETF，旨在提供与新兴市场互联网电子商务指数相对应的投资结果，该指数旨在衡量可投资的公开交易的新兴市场互联网和电子商务公司的表现。EMQQ里的公司有美团、拼多多、自由市场、腾讯。

截至2021年1月6日，EMQQ的价格为63.76美元，费用率为0.86%。自2014年11月12日成立以来，EMQQ提供了约160%的总回报，年均总回报约为17%。

电子商务在这里停留和增长。通过了解推动这一细分市场发展的主要公司，你可以进一步了解股票市场的主要增长点在哪里。记得继续做你的研究和尽职调查，尤其是如果你打算投资EBIZ、ONLN和EMQQ。阅读这篇文章很好，但如果能深入了解ETF所基于的指数、它们的顶级持股以及它们的管理团队就更好了。
