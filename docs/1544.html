我们将创建一个数组，输出从a到b的所有数字(我们的两个参数)。

今天，我们将编写一个名为between的函数，它将接受两个整数输入，a和b。

使用这两个数字作为我们的参数，该函数的目标是输出一个包含从a到b的所有数字的数组。这里有一个例子:

let a = 1;
let b = 4;// output: [1,2,3,4];

我们首先创建一个空数组，并将其赋给一个名为arr的变量。

let arr = [];

接下来，我们使用我们钟爱的for循环和从a到(包括)b的循环。每个数字进入arr数组。

for(let i = a; i <= b; i++){
   arr.push(i);
}

当我们完成时，我们返回我们的数组。

return arr;

以下是剩余的代码:

function between(a, b) {
  let arr = [];
  for(let i = a; i <= b; i++){
    arr.push(i);
  }
  return arr;
}
