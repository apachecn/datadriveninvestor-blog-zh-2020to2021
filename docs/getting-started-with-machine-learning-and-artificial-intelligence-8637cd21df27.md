# 不同类型的人工智能

1.  弱人工智能:-也被称为狭义人工智能，专注于一项任务。在弱人工智能的情况下，没有自我意识或真正的智能。
2.  强人工智能:-也被称为真正的人工智能，是一台像人脑一样聪明的计算机。这种人工智能将能够完成人类能做的所有任务。
