它需要三个参数。这三个参数中有两个是可选的。即start和step是可选参数。

  1. start 参数是序列的起始编号。即下限。默认情况下，如果没有指定，它从0开始。
  2. 停止参数是一个上限。即生成的数字不超过该数字，则range()在结果中不包括该数字。
  3. 第步是结果中每个数字之间的差值。如果未指定，步长的默认值为1。
  • 给定一个参数，它返回从到的整数值。例如，range(10)按顺序返回数字到。
  • 若要从以外的值开始，请使用两个参数调用range。例如，range(1，10)返回到之间的数字。
