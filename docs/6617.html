最后，我们来到了令人敬畏的python框架Django，它允许您使用Python开发web应用程序。

先决条件

首先确保您已经安装了python和pip。然后让我们安装Django:

python -m pip install Django

并检查它是否已安装并可以使用:

python -m django — version

如果你能看到Django库的一个版本，那么你已经准备好了。

项目设置

现在，让我们创建新的Django项目:

django-admin startproject myCoolProject
