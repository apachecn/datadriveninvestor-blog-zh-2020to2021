工作是你生活的70%。

如果你从事咨询行业，这是你必须接受的。作为一家财富500强公司的数据科学顾问，我每周向领导层汇报两次或更多。

初涉咨询业的人们心中有一个疑问:“我是否知道足够多的信息来建议别人下一步该怎么做？”

如果你是咨询的新手，你对咨询的了解还不够。

就这样了。你没有独立工作，这是你尚未看到的力量所在。咨询，无论是管理咨询还是技术咨询，就像你擅长的任何事情一样。你练习得越多，获得的经验越多，谈论和阅读得越多，你就会变得越好。

当我刚开始做咨询时，我对如何做事一无所知。作为一名新顾问，当你被要求交付成果时，你会处于一个复杂的空间。然而，当你逐渐地通过它，在你意识到之前，你已经在训练人们应该如何去做了！

好的公司文化给你一个指导方针和空间去试验，快速失败，学习和建立对你有效的流程。

当你养成了自己都没有注意到的小习惯时，最好记下你的做法并加以分析，因为这会给你一个优势。从行业中的中坚分子那里验证这些。

如果你的思维、行为和表现都像世界级的，你就是世界级的。

这是我作为一名管理顾问，在大流行后的世界中，记录我与来自不同水平的副总裁一起准备客户电话时所做的工作。

1.A计划和B计划(也许是C计划)

我确实想保持这个事实，但是想象力在我们的工作生活中起着非常重要的作用；比如，你认为客户电话会按照你的计划进行。😛

记住，仅仅因为你花了一周时间尝试做任务A或任务B，并不意味着客户对过程或结果感兴趣。如果他们想要任务C，那是你这边的失误或者沟通上的差距。句号。在这种情况下，保持冷静并准备好B计划至关重要。

此外，完全可以说这是你的失误，你会在会议记录(MoM)或下一次通话中获得更多信息。

从业务的角度，而不是从顾问/开发人员的角度来考虑客户演示。

对每个调用进行逆向工程。我们通过咨询来为业务做出贡献，而不是通过咨询来完成一周的任务然后回家。如果事情没有按照客户希望的那样发展，不要害怕站出来承认。团队可能会有最初的抵制，但是像你这样的人会站出来，相信我，在好的工作结束时总会有赞赏。否则，自讨苦吃。

2.陈述会议议程并帮助他们记录(使用幻灯片！)

抓住人们的注意力很难。帮助自己从会议议程开始，根据议程跟踪会议流程。

我学到的最好的东西之一是麦肯锡的3法则 ( 金字塔原理)

麦肯锡金字塔原则的关键要点是:

  1. 先从答案开始
  2. 3人一组，总结你的支持论点
  3. 对你的支持观点进行逻辑排序

总是在陈述会议议程时开始陈述。通过会议帮助观众了解我们完成了多少，还剩下多少。

例如，假设会议的议程有四个要点，而你在会议开始10分钟后即将开始第二个要点。不要说继续会议的第二个议程，试着说“对于4个议程项目中的第2个，我们将处理……”。

3.练习你的流畅状态(鼓励视频通话)

这可能是咨询中最关键的元素，也是最难掌握的一个。这在理论上似乎很容易掌握，但它要么来自敏锐地观察多个客户端调用，要么来自自己快速失败。

写下你在30分钟内最想交流的3件事。这有点像试图在不踩着新舞伴脚趾的情况下和他们跳舞。在这个大流行后的世界，你必须知道什么时候说话，什么时候积极倾听。使用“是的，还有……”的即兴方法。

随着你不断交流并逐渐建立融洽的关系，你会准确地知道在哪里停顿，学会阅读面部表情，并问一个问题或停顿一下。这都是一场表演，你是艺术家。

仅限视频通话。没有例外。即使你的客户不在视频上，如果你在的话，他们会觉得和你更有联系。你可以在福布斯网站上找到更多信息。

4.听取电话和MoM(与整个团队)的汇报

现在通话结束了！让我们进入下一个任务或者休息一下吧！

号码

趁大家还记忆犹新的时候，和你的团队一起分析这个电话是非常重要的，尤其是在大流行后的电话中。打电话听取汇报。详细讨论整个通话。将以下元素作为指导原则:

  1. 顺利的事情，不顺利的事情
  2. 客户对演示的反应
  3. 我们如何开始，如何结束——下次如何做得更好
  4. 会议的脉搏，以及我们是否能够有效地传达我们的信息
  5. 明显的讨论点、行动项目和预计到达时间

这有助于团队团结一致。这是一个很好的地方，让更多资深人士就该主题发表意见，以便团队中的年轻或新成员可以利用他们的经验，并在下次通话或未来利用这些经验。

5.八卦！(关于一切)

这似乎不像是一个传统的清单项目。在大流行后的世界中，我们所有人都有对流程、人员和任务的感知。仅仅通过30-60分钟的跨职能闲谈就可以获得大量信息。

鼓励你的团队谈论团队动态，提出恐惧，分享学习，谈论客户动态和工作效率。我认为这是一个非常有效的工具，特别是在大流行后的世界里，让每个人都感到联系更加紧密。

这导致了客户端调用性能的显著提高。怎么会？当演示者得到团队的信任时，他/她知道整个团队会支持他们，而不是一个人表演。类似地，团队更愿意提出新的想法来尝试，这使得咨询实践对每个人都更好，因此，最终成为一个更好的团队。
