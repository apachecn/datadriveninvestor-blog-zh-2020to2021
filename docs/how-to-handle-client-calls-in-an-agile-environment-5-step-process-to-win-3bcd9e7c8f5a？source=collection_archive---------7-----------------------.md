# 1.A 计划和 B 计划(也许是 C 计划)

我确实想保持这个事实，但是想象力在我们的工作生活中起着非常重要的作用；比如，你认为客户电话会按照你的计划进行。😛

记住，仅仅因为你花了一周时间尝试做任务 A 或任务 B，并不意味着客户对过程或结果感兴趣。如果他们想要任务 C，那是你这边的失误或者沟通上的差距。句号。在这种情况下，保持冷静并准备好 B 计划至关重要。

![](img/9f119f76f223c8cc32d2f9a9441c07ad.png)

Photo by [Kelly Sikkema](https://unsplash.com/@kellysikkema?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

此外，完全可以说这是你的失误，你会在会议记录(MoM)或下一次通话中获得更多信息。

> 从业务的角度，而不是从顾问/开发人员的角度来考虑客户演示。

**对每个调用进行逆向工程**。我们通过咨询来为业务做出贡献，而不是通过咨询来完成一周的任务然后回家。如果事情没有按照客户希望的那样发展，不要害怕站出来承认。团队可能会有最初的抵制，但是像你这样的人会站出来，相信我，在好的工作结束时总会有赞赏。否则，自讨苦吃。

# 2.陈述会议议程并帮助他们记录(使用幻灯片！)

抓住人们的注意力很难。**帮助自己**从会议议程开始，根据议程跟踪会议流程。

我学到的最好的东西之一是[麦肯锡的 3 法则](https://medium.com/p/3bcd9e7c8f5a/edit) ( [金字塔原理](https://medium.com/lessons-from-mckinsey/the-pyramid-principle-f0885dd3c5c7))

![](img/848e81a722b1f9dd79997873d7453063.png)

Photo by [Campaign Creators](https://unsplash.com/@campaign_creators?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

麦肯锡金字塔原则的关键要点是:

1.  先从答案开始
2.  3 人一组，总结你的支持论点
3.  对你的支持观点进行逻辑排序

总是在陈述会议议程时开始陈述。通过会议帮助观众了解我们完成了多少，还剩下多少。

例如，假设会议的议程有四个要点，而你在会议开始 10 分钟后即将开始第二个要点。不要说继续会议的第二个议程，试着说“对于 4 个议程项目中的第 2 个，我们将处理……”。

# 3.练习你的流畅状态(鼓励视频通话)

![](img/74f932fc1a9fa80fa5b304658a9bb54b.png)

Video Calls only. (Credits: [Gabriel Benois](https://unsplash.com/@gabrielbenois?utm_source=medium&utm_medium=referral))

这可能是咨询中最关键的元素，也是最难掌握的一个。这在理论上似乎很容易掌握，但它要么来自敏锐地观察多个客户端调用，要么来自自己快速失败。

写下你在 30 分钟内最想交流的 3 件事。这有点像试图在不踩着新舞伴脚趾的情况下和他们跳舞。在这个大流行后的世界，你必须知道什么时候说话，什么时候积极倾听。使用[“是的，还有……”的即兴方法](https://medium.com/@kenmacken/yes-but-why-not-yes-and-5e62761f27ca)。

随着你不断交流并逐渐建立融洽的关系，你会准确地知道在哪里停顿，学会阅读面部表情，并问一个问题或停顿一下。这都是一场表演，你是艺术家。

**仅限视频通话。没有例外。即使你的客户不在视频上，如果你在的话，他们会觉得和你更有联系。你可以在福布斯网站上找到更多信息。**

# 4.听取电话和 MoM(与整个团队)的汇报

现在通话结束了！让我们进入下一个任务或者休息一下吧！

号码

![](img/d99571f6e4f1b2c05647fb3eaca69474.png)

Debrief Call (Credits [Antenna](https://unsplash.com/@antenna?utm_source=medium&utm_medium=referral))

趁大家还记忆犹新的时候，和你的团队一起分析这个电话是非常重要的，尤其是在大流行后的电话中。**打电话听取汇报**。详细讨论整个通话。将以下元素作为指导原则:

1.  顺利的事情，不顺利的事情
2.  客户对演示的反应
3.  我们如何开始，如何结束——下次如何做得更好
4.  会议的脉搏，以及我们是否能够有效地传达我们的信息
5.  明显的讨论点、行动项目和预计到达时间

这有助于团队团结一致。这是一个很好的地方，让更多资深人士就该主题发表意见，以便团队中的年轻或新成员可以利用他们的经验，并在下次通话或未来利用这些经验。
