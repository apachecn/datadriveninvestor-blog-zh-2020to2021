Data miner是一个扩展工具，可以在谷歌Chrome和微软Edge上运行。它可以帮助你将网页中的数据抓取到CSV文件或Excel电子表格中。有许多定制的方法可用于抓取amazon数据。如果那些提供的正是你所需要的，这可能是一个方便的工具，你只需点击几下就可以从亚马逊上找到。

数据矿工有一个循序渐进的友好界面和基本功能的网页抓取。它更适合小型企业或休闲使用。

Data Miner的免费计划有页面限制(500/月)。如果需要多刮，专业的和其他付费的计划都有。

卷筒纸刮刀

Web Scraper是一个扩展工具，在开发工具中集成了一个点击界面。如果没有电子商务或亚马逊抓取的特定模板，您必须通过在网页上选择您想要的列表信息来构建自己的爬虫。

Web scraper配备了云提取、定时抓取、IP轮换、API访问等功能(可用于付费计划)。因此，它能够更频繁地抓取和抓取更大量的信息。

刮刀分析器

Scraper Parsers是一款浏览器扩展工具，用于提取非结构化数据，无需代码即可可视化。提取的数据可以在网站上查看或以各种形式下载(XLSX、XLS、XML、CSV)。提取数据后，数字可以相应地显示在图表中。

解析器的UI是一个面板，你可以在浏览器上拖动并通过点击来选择，它还支持预定的抓取。然而，它似乎不够稳定，很容易卡住。对于一个访问者来说，每个网站的使用限制是600页。报名的话可以多拿590。

这个试用版只能下载2页的任何搜索查询。您需要购买完整版来下载无限的页面，并获得1年的免费支持。
