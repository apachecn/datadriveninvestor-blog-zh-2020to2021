# 什么是 TypeScript？

Typescript 是一种由微软开发和维护的编程语言。它是 JavaScript 语言的一个严格的语法超集，并向该语言添加了额外的静态类型。TypeScript 由微软设计，用于大型应用程序的开发，它可以转换为 JavaScript。
