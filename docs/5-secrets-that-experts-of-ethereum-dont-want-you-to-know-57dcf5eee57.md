# 什么是松露？

Truffle 是以太坊的开发环境、测试框架和资产管道，旨在让以太坊开发者的生活更加轻松。有了松露，你会得到:

*   内置智能合约编译、链接、部署和二进制管理。
*   Mocha 和 Chai 的自动化合约测试。
*   支持自定义构建流程的可配置构建管道。
*   可脚本化的部署和迁移框架。
*   部署到许多公共和私有网络的网络管理。
*   用于直接合同通信的交互式控制台。
*   开发过程中资产的即时重建。
*   在 Truffle 环境中执行脚本的外部脚本运行程序。
