2020年8月22日，南非云开发者倡导者Sbusiso Mkhombe & UAE云开发者倡导者 Mridul Bhandari 举办了一场网络研讨会，主题是如何使用Red Hat Universal Base Image在OpenShift上部署一个货币兑换示例应用程序。

总共有5名与会者，包括来自MEA的开发人员和学生。网上研讨会以向观众介绍 IBM Developer 系列的欢迎辞开始，分为两个主要部分。第一部分是向观众简要介绍架构、虚拟机、容器、RedHat OpenShift，另一部分是代码模式。

第一部分由Sbusiso Mkhombe领导。他解释了不同的架构，虚拟机和容器之间的区别，以及RedHat OpenShift。

网络研讨会的第二部分由Mridul Bhandari主持，旨在使用Red Hat Universal Base Image向OpenShift部署一个示例货币兑换应用程序。他首先向与会者展示如何在本地部署Docker映像。然后他解释了什么是UBI和不同的基础图像。然后将应用部署到RedHat OpenShift集群。

在网上研讨会结束时，与会者被介绍给了 IBM开发人员资源。许多与会者对网上研讨会表现出积极的反馈和赞赏。
