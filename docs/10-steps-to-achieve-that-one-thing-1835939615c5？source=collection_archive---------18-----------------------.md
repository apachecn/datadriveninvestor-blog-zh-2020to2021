# 1.没有远大的目标会限制你的机会。

当 J.K .罗琳构思《哈利·波特》的时候，她甚至在写第一部的第一章之前就设想了七本关于霍格沃茨生活的书。

她不怕想得太大:甚至在开始朝着成功努力之前，就从一个宏大的成功愿景开始。很难想象如果她当初没有如此突出地思考，她会取得如此巨大的成功。

然而，对大多数人来说，伟大的想法或显著的成就令人生畏，并有负面的联想，比如感觉*不知所措*和*被吓倒*。这些消极的想法通常会阻碍人们的大局观。

当我们不能从大处着眼，让这些负面的联想支配我们时，我们的思维会萎缩，我们会降低我们的轨迹。我们积极地限制我们潜在的成就，让自己变得平庸。

成功需要行动，行动需要思考。但是要取得非凡的成果，我们的努力首先必须基于大思维。

# 2.区分待办事项的优先顺序——它们并不都同样重要。

大多数人会时不时地列出“待办事项”清单来记录他们必须完成的所有任务。但是一旦你有了清单，你如何决定先做哪一项呢？

你是从最耗时的任务开始，还是先检查较小的任务？也许你是按照书写的顺序来完成的？

这些方法没有解决一个关键点:所有的项目并不同等重要。

其中只有少数可能会产生深远的影响；他们应该得到最优先的考虑。

你可能听说过 80/20 原则:80%的结果或产出总是由 20%的工作或投入来完成。

这个原则的含义很清楚:你的待办事项清单上的任务并不同等重要；其中的一小部分将对你的成功做出最大的贡献。对你的任务进行优先排序，把注意力集中在那些能取得最大成效的任务上。

# 3.问“聚焦问题”将帮助你分清轻重缓急，创建可行的任务，并实现你的目标。

关于成功，马克·吐温曾经说过:

“取得成功的秘诀在于开始行动。开始行动的秘诀是把复杂的、势不可挡的任务分解成小的、容易处理的任务，然后从第一个任务开始。”

“我能做的一件事是什么，通过做这件事，其他一切都会变得更容易或不必要？”

这个问题可以在两个层面上提出，每个层面都有其功能:

首先，在宏观层面上，聚焦问题可以帮助你看到*大图*并确定你的总体目标:*你一生中想做并实现的一件事*。例如，在这里，你的一件事可能是你的整体职业目标。

第二，在更实际的短期层面上，聚焦问题为你提供了一个*小焦点*，让你优先考虑眼前的选项，选择最重要的任务开始。你在这里寻找*你现在能做的一件事*；例如，“打那个电话。”

第一个层次是关于找到正确的人生方向；第二是选择正确的行动。

# 4.自律生活的秘诀是循序渐进地养成习惯。

当我们想到像比尔·盖茨一样成功的人时，我们倾向于将他非凡的成功归因于非凡的自律，这种自律使他在性格形成时期学会了如何编写计算机程序。这种纪律水平似乎是一个令人生畏的无法实现的目标。成功人士是如何保持这样的纪律性的？

一个习惯只是开始。保持习惯比开始习惯要容易得多，所以一旦某样东西变成了习惯，你可以把你的训练转变成形成一个新的习惯，然后循序渐进地建立起来。例如，你可以每天提前 30 分钟进入办公室，在你的同事到达之前清理你的收件箱。一旦养成了这个习惯，你就可以通过引导你的纪律在更长的时间内专注于一个特定的任务来建立这个习惯。一旦养成了这个习惯，你就可以进入下一个了。

通过有选择地应用纪律来建立积极的习惯，会给你一个纪律严明的生活的外观和好处，而不需要超人的纪律。

# 5.多任务处理的效率低得可怕:选择一件事，并全神贯注于它。

如今，人们普遍认为多任务处理是一件有效的事情。我们通常理解这个术语的意思是:同时做两件或更多的事情。但它最初被创造出来是为了描述一台计算机使用一个处理器处理多项任务，*在它们之间快速连续地来回交替*——这一区别非常有启发性。

通常，当我们认为我们正在进行多任务处理时，我们正在处理两个或更多的任务，就像电脑一样，将注意力从一个转移到另一个。

研究表明，对于人类来说，这种任务转换需要时间惩罚，因为从一项任务转移到另一项任务需要时间。在相对简单的任务中，这种时间成本可能很小，但当你要返回的任务更复杂时，这种时间成本就会大大增加。

# 6.你的意志力就像一个油箱:仔细选择你用它的地方，否则当你真正需要它的时候你可能会用完它。

大多数人痛苦地意识到他们没有坚定的意志力。然而，令人惊讶的是，研究表明，我们的意志力远不是一种恒定的资源，实际上在一天中以不同的速度消耗，这取决于我们从事的活动。

例如，当我们决定集中注意力、抑制情绪或改变行为以追求目标时，我们的意志力就会耗尽。当我们的意志力被耗尽时，我们就不能发挥它，如果进一步的任务需要这种资源的话。

这就解释了为什么在做了一段时间艰难的决定或从事了一段时间乏味的工作后，你可能无法抗拒美味的小吃。

全力以赴的意志力需要一个满满的油箱，所以计划好你的一天，这样你就可以避免在快用完的时候做出关键的决定或判断。

# 7.如果你想把精力集中在最重要的任务上，对不重要的任务说不是至关重要的。

我们都曾因为想要帮助别人而拒绝别人的请求。帮助别人会有很大的回报，但是，为了节省你的时间和精力来实现你最大的目标，你需要对低优先级的请求说不。

众所周知，史蒂夫·乔布斯对他没有追求的项目和他追求的项目一样自豪。当他 1997 年回到苹果公司时，他将公司的产品从 350 种减少到只有 10 种。这是很多的不。

一直对别人说不并不一定要像看起来那样冷漠或自私。你可以试着给他们一个不需要你帮助的替代解决方案，或者把他们介绍给更有帮助的人。

在时间和资源有限的情况下，如果你想集中精力完成最重要的任务，你必须准备好对琐碎的任务说不。

# 8.永远不要为了你的职业目标而牺牲你的个人生活——相反，无情地优先考虑你的工作时间。

我们都努力实现平衡的生活——在所有对我们重要的事情上平均分配我们的时间——但我们这样做时没有停下来问为什么，以及我们实际上想要实现的是什么。

可以理解的是，我们努力寻求平衡，因为我们工作生活和个人生活中的所有需求都显得如此重要。然而，这种平衡生活的愿景是得不到的，也是不可取的。

诀窍是无情地优先安排你的工作时间，以专注于职业目标。在你的个人生活中，忽视任何一个方面都是危险的；但是你在工作生活中有你的头等大事，其他的都可以商量。有时，不太重要的事情将不得不被最小化，被等待或被其他人管理，直到重要的事情被完成。这种方法会让你专注于你最重要的工作，给它所需要的集中力来取得伟大的成果。

# 9.为了专注于你的一件事，你需要有效的时间管理策略，并接受其他领域的一些混乱。

安排一些时间来做你的一件事，对它们做出承诺，并捍卫它们，就像它们是你最重要的约会一样。你还需要确保你的物理环境不会妨碍你有效地利用这段时间。无论你在哪里工作，你都需要尽量减少潜在的干扰。如果不可能的话，考虑在办公室以外的地方工作。

诸如此类的技巧可以让你给你的事情以应有的关注。

# 10.带着目标生活，想象实现目标的步骤，这会让你走上通往非凡结果的道路。

有了目标，你就可以开始计划实现目标的步骤，但如果你能想象出前进道路上的每一步，那就更好了，因为这会激励你，为你的未来做准备。这些有益的效果在一项对学生的研究中得到了展示，当他们参加考试时，被要求想象考试的结果或者准备和参加考试的过程。想象这一过程的学生报告了更高的动机水平，准备更充分，随后取得了更好的结果。

你的目标突然清晰起来:这是你想做的一件事。如果你现在做一些研究，开始训练，并获得合适的设备，那将是最好的。感谢你的目标，你已经在制定计划，设想步骤，并朝着目标前进。

定义一个目标——我们在生活中想要做和实现的一件事——是我们都应该努力的事情。

# 最后的话-

**成功来自专注于一件事，而不是许多事情。当朝着你的目标努力时，避免那些阻碍你成功的陷阱。学习如何从杂乱中理出头绪，在真正重要的地方做最好的工作。**
