# **4 着色机**

**C***olor izer***是一个使用 Lua 用生成式对抗网络(GANs)为黑白图像添加色彩的项目。**

**该项目旨在 ***给黑白图像添加色彩。即，发生器网络(G)接收其黑白版本，并输出图像的全 RGB 版本*****
